Course: Web Development

Project Description:
MediaVault is a web-based media sharing platform that allows users to upload,
manage, and view images and videos. Users can search media, edit metadata,
and manage their personal uploads.

Technologies Used:
HTML5
CSS3
JavaScript
LocalStorage (for demo data storage)

Website Pages:
index.html – Home page displaying trending media and user uploads.
upload.html – Allows users to upload new media files.
my-media.html – Shows all media uploaded by the current user.
profile.html – Handles username login and profile settings.
details.html – Displays detailed information about a media item.
edit.html – Allows users to edit metadata such as title, description, and tags.

Metadata Implementation:
Meta tags were added to each HTML page in the head section including:
- description
- keywords
- author
- robots
- Open Graph tags
- Twitter card tags

These metadata elements help search engines understand the content of the
website and improve accessibility and SEO.
